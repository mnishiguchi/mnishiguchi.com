```html
<div class="container">
  <div class="left">
    <!-- left column -->
  </div>
  <div class="contents">
    <!-- right column -->
  </div>
</div>
```

```css
.container{
  display: flex;
}
.left{
  width: 300px;
}
.contents{
  flex: 1;
}
```

#### `flex` property
> [a shorthand property that sets flex-grow, flex-shrink, and flex-basis](https://developer.mozilla.org/en-US/docs/Web/CSS/flex)
